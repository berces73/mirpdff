Frequently Asked Questions

Answers to your questions about MirPDF

🔒 Security & Privacy

Is MirPDF safe? Are my files uploaded to a server?

It depends on the tool. MirPDF uses a hybrid architecture: some tools run entirely in your browser, while others require server-side processing.

🟢 Browser-only tools (files never leave your device):

  • Merge PDF, Split PDF, Rotate, Delete Pages, Reorder Pages
  • JPG → PDF conversion
  • Add Watermark, Password Protect

🔵 Server-side tools (HTTPS + auto-delete within 1 hour):

  • OCR (text recognition) — processed on server, deleted within 1 hour
  • Compress PDF — processed on server, deleted within 1 hour
  • PDF → Word conversion — processed on server, deleted within 1 hour

All files sent to our servers use HTTPS encryption and are automatically deleted within 1 hour. No files are stored permanently.

What data do you collect?

We collect minimal data:

  • ✅ Page views (Google Analytics)
  • ✅ Browser type and operating system
  • ❌ File contents — not retained permanently
  • ❌ File names — not logged
  • ❌ Personal information — no account system

Can I use MirPDF with sensitive documents?

For browser-based tools: Yes — your file never reaches our servers, making these fully private.

For OCR, Compress, and PDF-to-Word: Files are sent over HTTPS and deleted within 1 hour. For highly sensitive documents (medical records, legal files, financial data), we recommend using browser-based tools or a local application.

⚠️ MirPDF does not hold HIPAA or equivalent health data compliance certifications. If you handle health records, use a certified enterprise solution.

🛠️ Features & Usage

Do I need to create an account?

No, there is no account system. All tools are free and open to everyone. No name, email, or registration required.

Is there a file size limit?

It depends on the tool and your device memory.

  • 💻 Desktop: 500 MB – 2 GB typically works without issues
  • 📱 Mobile: 50–200 MB recommended
  • 🐏 RAM-dependent: 100–200 MB ideal on 4 GB RAM

Can I use MirPDF offline?

Partially. Browser-based tools (merge, split, rotate, etc.) can work offline after the initial load thanks to Service Worker caching. OCR, Compress, and PDF-to-Word tools require an internet connection as they run on our servers.

Which browsers are supported?

All modern browsers are supported:

  • ✅ Chrome (90+)
  • ✅ Firefox (88+)
  • ✅ Safari (14+)
  • ✅ Edge (90+)
  • ❌ Internet Explorer (not supported)
