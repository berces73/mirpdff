Frequently Asked Questions

Is cropping applied to all pages?

You can apply to all pages or specify a page range.

Is cropped content permanently deleted?

Technically no — PDF cropping adjusts the visible area. Original content is preserved in the file structure.

Can I undo cropping?

You can re-upload and crop with different values. Keep your original file as a backup.

Can I crop different amounts from different pages?

Currently, the same crop values apply to all selected pages. For page-specific cropping, process pages in separate batches.

Will cropping affect text searchability?

No. Cropping only changes the visible area — text remains searchable in the output PDF.

Can I uncrop a PDF that someone else cropped?

If the PDF was originally cropped by this or similar tools, a PDF editor may be able to restore the hidden content by resetting crop values.

When to Use This Tool

Scanner border removal
Flatbed scanners often add black or white borders. Crop to remove these and get clean, professional documents.
E-reader optimization
Reduce margins on academic PDFs or e-books for better readability on tablets and e-ink readers.
Focusing on a table or chart
Extract just the relevant data visualization from a large page for presentations or reports.
Standardizing document margins
Ensure consistent margins across a batch of documents before merging into a single file.
💡 Pro Tips
  • Crop values are in points (pt) by default. 72pt = 1 inch = approximately 2.54 cm.
  • Apply a small test crop first to verify the values before processing the full document.
  • Cropping hides but doesn't permanently delete content — the original data is still in the file structure.
  • After cropping, use Compress PDF to reduce file size if the output is larger than expected.

When to Crop PDF Pages

Removing Scanner Borders
Flatbed scanners add black or grey borders. Crop equally from all four sides to produce a clean, border-free scan.
Isolating a Chart or Table
A single chart or table embedded in a larger report page can be cropped into its own focused PDF for presentations or reports.
E-Reader Optimisation
Technical PDFs with wide margins waste screen space on tablets and e-readers. Crop to narrow margins for a larger, more readable text area.
Standardising Mixed Margin Documents
Documents assembled from multiple sources often have inconsistent margins. Crop to a uniform visible area for a professional, consistent appearance.

Pro Tips

  • Crop symmetrically (equal values on left and right) to keep text centred on the page.
  • Cropping hides — not deletes — content outside the visible area. The hidden content is technically still in the file.
  • For scanner border removal, a 5-10px crop from each side usually eliminates the border without touching document content.
  • Preview at 100% zoom before finalising — small crops are easy to miscalculate at reduced zoom levels.
