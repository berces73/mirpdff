Frequently Asked Questions

What encryption level is used?

AES-128 or AES-256 — the same standard used in banking and government systems.

What if I forget the password?

There is no recovery option — only you know the password. Store it safely.

Can I remove the password later?

Yes. Use our Unlock PDF tool with the correct password to remove protection.

What encryption standard does MirPDF use for locking?

AES-256 — the same standard used by banks, governments and military applications worldwide.

Can a locked PDF be unlocked without the password?

AES-256 encrypted PDFs are practically unbreakable with a strong password. Weak passwords are vulnerable to brute-force attacks.

Does locking affect the visual appearance of the PDF?

No. Password protection is invisible — the PDF looks and reads identically when opened with the correct password.

💡 Pro Tips
  • Use a strong password: 12+ characters, mix of letters, numbers and symbols. Avoid birthdays or names.
  • Never include the password in the same email as the PDF — send via a separate channel (SMS, phone call).
  • Store passwords in a password manager, not in a text file or sticky note.
  • If you need recipients to fill in a form, use Edit PDF first, then lock — locking prevents further edits.

When to Password-Protect a PDF

Confidential Business Documents
Financial reports, HR files, board minutes — protect sensitive documents before emailing. Even if forwarded, recipients without the password cannot open the file.
Client Deliverables
Consultants and agencies send proposals and reports to clients. Password-protect to control access — only the intended recipient can open the document.
Personal Records
Tax returns, medical records, bank statements — protect personal PDF archives with a password to guard against unauthorised access if a device is lost or shared.
Preventing Unauthorised Editing
Legal contracts and official templates should not be editable. Set an owner password with editing restrictions — recipients can read but not modify.

Pro Tips

  • Use a unique password per document — a single compromised password shouldn't unlock your entire document archive.
  • AES-256 encryption is practically unbreakable. The weak point is always the password itself — use 12+ characters with mixed types.
  • Send the password via a different channel than the PDF — phone, separate email, or messaging app.
  • Store passwords in a password manager (Bitwarden, 1Password) — never in a text file or email.
