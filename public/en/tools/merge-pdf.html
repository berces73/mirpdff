Frequently Asked Questions

Is merging PDF files free?

Yes, completely free. Merging runs in your browser with no server cost — no registration required, no time limits.

Can I merge password-protected PDFs?

You'll need to unlock them first using our PDF Unlock tool, then merge.

Why did the merged file get larger?

Each page retains its original quality. To reduce size, run the merged file through our Compress PDF tool afterwards.

What is the maximum number of pages?

Practically limited by your browser's RAM. Most computers handle 200–300 page merges without issues.

Can I merge PDFs with different page sizes (A4 + Letter)?

Yes. Different page sizes are preserved — each page retains its original dimensions in the merged output.

Does merging change image or text quality?

No. Merging is a structural operation — no re-rendering occurs, original quality is preserved throughout.

Can I merge more than 20 files at once?

Free plan: up to 20 files per merge. Pro plan removes this limit.
