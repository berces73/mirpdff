Frequently Asked Questions

Can I skip the first page?

Yes. Set the starting page to 2 or higher to leave the cover page unnumbered.

Can I use Roman numerals?

Standard Arabic numerals are supported. Roman numeral support coming soon.

Can page numbers be removed later?

Page numbers added by this tool are embedded as PDF content — removal requires a PDF editor.

When to Add Page Numbers

Legal & Formal Documents
Contracts, court filings, and official reports require page numbers for reference during review and signature. Numbered pages prevent disputes about which version was reviewed.
Printed Document Navigation
Long printed manuals and books without page numbers are unusable for reference. Numbering is essential for any document where readers need to locate specific sections.
Document Assembly
After merging multiple PDFs, the combined document needs consistent page numbering throughout. Add numbers after merging.
Meeting Materials
Agendas and board packs distributed before meetings need page numbers so all participants can reference the same page during discussion.

Pro Tips

  • Skip the first page (cover) by setting start page to 2 — keeps the cover unnumbered while numbering everything else.
  • Footer center placement is the most universally readable position for page numbers.
  • For documents continuing a series, set a custom start number (e.g. page 47) to maintain continuity across volumes.
  • Add page numbers as the final step — after merging, reordering, and editing — so numbers reflect the final page sequence.
